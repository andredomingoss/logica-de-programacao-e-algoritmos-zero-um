1 - Faça um algoritmo que determine o maior número entre N números. A condição de parada é a entrada de um valor 0, ou seja, o algoritmo deve ficar calculando o maior até que a entrada seja igual a 0 (ZERO).
Algoritmo "semnome"
// Disciplina   : [Linguagem e Lógica de Programação]
// Autor(a)    : André Domingos

Var
valor, maior: inteiro


Inicio
repita
    escreval("Digite um valor: ")
    leia(valor)
    
    se(valor > maior) entao
        maior <- valor
    fimse
ate(valor = 0)

escreval("O maior valor digitado: ", maior)

Algoritmo "semnome"
// Disciplina   : [Linguagem e Lógica de Programação]
// Autor(a)    : André Domingos

Var
contador, valor, multiplo: inteiro


Inicio
contador <- 1
valor <- 100
multiplo <- 10

enquanto (contador <= valor) faca
   escreval(contador)
   
   se(contador mod multiplo = 0) entao
      escreval("O ", contador, " é multiplo de 10!")
   fimse

   contador <- contador + 1
fimenquanto

5 - Faça um algoritmo que receba 5 valores, e exiba a: 
* Soma dos valores
* media entre os valores
* quantos sao divisiveis por 5
* quantos sao nulos
* soma dos pares
Algoritmo "semnome"
// Disciplina   : [Linguagem e Lógica de Programação]
// Autor(a)    : André Domingos

Var
valor : vetor [0..4] de inteiro
contador, soma, par, divisor: inteiro
media: real

Inicio
divisor <- 5
para contador de 0 ate 4 faca
   escreval("Entre com um número:")
   leia(valor[contador])
   
   soma <- soma + valor[contador])
   media <- soma / 5

   se (valor[contador] % 2 = 0) entao
       par <- par + valor[contador]
   fimse
   
   se(valor[contador] mod divisor = 0) entao
        escreval("O ",valor[contador]," é divisível por 5!")
   fimse
   
fimpara
escreval("========================")
escreval("A soma dos valores é: ", soma)
escreval("A média  dos valores é: ", media)
escreval("A soma dos valores pares é: ", par)

• Encontre o maior valor
• Encontre o menor valor

Algoritmo "semnome"
// Disciplina   : [Linguagem e Lógica de Programação]
// Autor(a)    : André Domingos

Var
quantidade, contador, numero, maior, menor, soma: inteiro


Inicio
escreval("Digite a quantidade de valores que você quer inserir: ")
leia(quantidade)

contador <- 1

enquanto (contador <= quantidade) faca
   escreval("Digite o valor do número ", contador, ":")
   leia(numero)
   
   se(numero > maior) entao
     maior <- numero
   senao
     se(numero < maior) entao
       menor <- numero
     fimse
   fimse
   
   contador <- contador + 1

fimenquanto

escreval("O menor valor digitado foi:", menor)
